# Drones Shop

**Drones Shop** — це навчальний повнофункціональний інтернет-магазин, розроблений на Flask. Проєкт включає каталог товарів, кошик, оформлення замовлень, авторизацію користувачів, інтеграцію сторонніх API та систему міграцій бази даних. Проєкт виконаний у рамках навчання та демонструє повний цикл створення веб‑застосунку.

## Використані технології

### Backend
- **Python**
- **Flask** — основний веб‑фреймворк
- **Flask‑Login** — система авторизації
- **Flask‑SQLAlchemy** — ORM для роботи з БД
- **Flask‑Migrate** — міграції
- **SQLite** — база даних
- **python‑dotenv** — робота з .env

### Frontend
- HTML5
- CSS3
- Jinja2

## Установка та запуск

1. Клонування репозиторію:
```
git clone https://github.com/WorldIT-academy/Drones-Shop-17-30.git
cd Drones-Shop-17-30
```

2. Встановлення залежностей:
```
pip install -r requirements.txt
```

3. Створення `.env` файлу
```
touch .env
```

4. Запуск проєкту:
```
python manage.py
```

Сервер буде доступний за адресою:  
http://127.0.0.1:8001
